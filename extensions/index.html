Microsoft Rewards Helper

A small extension that helps speed up the completion of the microsoft rewards program by autocompleting and providing hints for quizzes.

Download zip file

How to install

  1. Download and extract the zip file to an empty folder
  2. Navigate to the extension page in your browser (about://extensions in Chromium browsers such as Chrome or Edge)
  3. Make sure 'Developer mode' is toggled on
  4. Click on the 'Load unpacked' button and select the folder with the files you extracted earlier
  5. Use the extension!
