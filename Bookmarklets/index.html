Bookmarklets

Bookmarklets are bookmarks that execute JavaScript when clicked on, instead of opening a page. They can be used for a wide variety of small tasks that can add features or increase productivity.

My Bookmarklets

    Wall Zapper - Remove annoying paywalls and subscribe walls.

    Distraction Free YouTube - Remove all the distractions from YouTube, leaving only the video.

    Joke Generator - A basic joke generator. Note that this doesn't actually generate jokes, but rather pulls them from a human-curated database.

    Edit Page - Edit any webpage.
